/post/new : create a new post with a time stamp and title. Links to the profile_id.
/comment/new: create a comment with a time stamp and an id that connects it to the post.
/profile/name: Display a profile(via a object, read_only)
/profile create/new : create a profile
/user/new : register a user and have them create a profile (prompts profile/create)
/login : Logs the user with a username and password
/profile/update : Update/edits the profile
/profile/attributes/update : updates the attributes values (all ints)
/post/delete : deletes post
/comment/delete : deletes comment
user/profile/delete: deletes profile(including anything linked to thier profile id: comments, posts, etc)
and user from system, executes an alert to warn user first.
