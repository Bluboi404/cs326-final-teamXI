v/post/new : create a new post with a time stamp and title. Links to the profile_id.
v/comment/new: create a comment with a time stamp and an id that connects it to the post.
v/profile/name: Display a profile(via a object, read_only)
v/profile/new : create a profile
v/user/new : register a user and have them create a profile (prompts profile/create)
v/login : Logs the user with a username and password
v/profile/edit : Update/edits the profile
v/profile/attributes/update : updates the attributes values (all ints)
v/post/delete : deletes post
/comment/delete : deletes comment
v/user/profile/delete: deletes profile(including anything linked to thier profile id: comments, posts, etc)
and user from system, executes an alert to warn user first.
